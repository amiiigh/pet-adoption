# Server
`$ git clone https://github.com/amiiigh/pet-adoption`

`$ cd pet-adoption/server`

`$ node server.js`


# Client
`$ git clone https://github.com/amiiigh/pet-adoption`

`$ cd pet-adoption/client`

`$ npm install`

`$ npm run start`

`pet/server/animals.json` contains the sample pets information.
